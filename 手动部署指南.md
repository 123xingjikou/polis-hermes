# QClaw 进程优化 - 手动部署指南

## 一、agent_supervisor 手动部署

### 步骤 1：复制文件
将 agent_supervisor.py 复制到 .qclaw 目录：

```powershell
# 在 PowerShell 中执行（以普通用户身份）
Copy-Item "C:\Users\dfhai\AppData\Roaming\TRAE SOLO CN\ModularData\ai-agent\work-mode-projects\6a45044b9f3d6718577ed1f8\agent_supervisor.py" "C:\Users\dfhai\.qclaw\workspace-hermes\agents\agent_supervisor.py" -Force
```

### 步骤 2：更新 guardian.bat
修改 `C:\Users\dfhai\.qclaw\workspace-hermes\guardian.bat`，将原来的3个独立agent监控改为监控agent_supervisor：

**原来的内容（第14-16行）：**
```batch
call :check "trae_agent"     "agents\trae_agent.py" ""
call :check "opencode_agent" "agents\opencode_agent.py" ""
call :check "agent_coordinator" "agents\agent_coordinator.py" ""
```

**改为：**
```batch
call :check "agent_supervisor" "agents\agent_supervisor.py" ""
```

### 步骤 3：启动 agent_supervisor
有两种方式：

**方式 A：通过 guardian.bat 启动**
```powershell
# 启动 guardian（它会自动监控和启动 agent_supervisor）
cd C:\Users\dfhai\.qclaw\workspace-hermes
Start-Process guardian.bat -WindowStyle Hidden
```

**方式 B：直接启动**
```powershell
cd C:\Users\dfhai\.qclaw\workspace-hermes
$pyw = "C:\Users\dfhai\AppData\Local\Programs\Python\Python313\pythonw.exe"
Start-Process $pyw -ArgumentList "agents\agent_supervisor.py" -WindowStyle Hidden
```

### 步骤 4：验证
```powershell
# 检查进程
Get-Process pythonw | Where-Object { (Get-CimInstance Win32_Process -Filter "ProcessId=$($_.Id)").CommandLine -like "*agent_supervisor*" }
```

## 二、系统服务禁用（需管理员权限）

### 需要以管理员身份运行 PowerShell
右键点击 PowerShell → "以管理员身份运行"

### 安全可禁用的服务列表

| 服务名 | 显示名 | 说明 | 禁用命令 |
|--------|--------|------|----------|
| WSearch | Windows Search | 禁用后搜索变慢，但不影响基本功能 | `Stop-Service WSearch -Force; Set-Service WSearch -StartupType Disabled` |
| DiagTrack | Connected User Experiences and Telemetry | 遥测数据收集，隐私相关 | `Stop-Service DiagTrack -Force; Set-Service DiagTrack -StartupType Disabled` |
| SysMain | SysMain (Superfetch) | 内存预读，禁用可减少磁盘IO | `Stop-Service SysMain -Force; Set-Service SysMain -StartupType Disabled` |
| lfsvc | Geolocation Service | 地理位置，除非需要定位功能 | `Stop-Service lfsvc -Force; Set-Service lfsvc -StartupType Disabled` |
| PcaSvc | Program Compatibility Assistant | 程序兼容性提示 | `Stop-Service PcaSvc -Force; Set-Service PcaSvc -StartupType Disabled` |
| CDPSvc | Connected Devices Platform | 连接设备平台，跨设备同步 | `Stop-Service CDPSvc -Force; Set-Service CDPSvc -StartupType Disabled` |
| Themes | Themes | 主题服务，禁用后变成经典外观 | `Stop-Service Themes -Force; Set-Service Themes -StartupType Disabled` |
| NcbService | Network Connection Broker | 网络连接代理 | `Stop-Service NcbService -Force; Set-Service NcbService -StartupType Disabled` |

### 一键禁用脚本（管理员 PowerShell）
```powershell
$services = @('WSearch', 'DiagTrack', 'SysMain', 'lfsvc', 'PcaSvc', 'CDPSvc', 'Themes', 'NcbService')
foreach ($svc in $services) {
    try {
        Stop-Service $svc -Force -ErrorAction SilentlyContinue
        Set-Service $svc -StartupType Disabled
        Write-Host "已禁用: $svc" -ForegroundColor Green
    } catch {
        Write-Host "禁用失败: $svc - $_" -ForegroundColor Yellow
    }
}
```

### 不建议禁用的服务
- **BITS** - 后台智能传输，Windows更新需要
- **wuauserv** - Windows Update，系统更新需要
- **RemoteRegistry** - 已禁用
- **Fax** - 已禁用
- **RetailDemo** - 已禁用

## 三、预期效果

### 进程变化
- 原来：4个核心进程 + 3个agent进程 = 7个 Pythonw 进程
- 优化后：4个核心进程 + 1个agent_supervisor进程 = 5个 Pythonw 进程
- 减少：2个进程，节省约 10-15 MB 内存

### 服务禁用效果
- 减少后台服务占用
- 降低磁盘IO（禁用WSearch和SysMain）
- 减少遥测数据上传
